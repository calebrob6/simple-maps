# simple-maps
Python library for easily creating maps from a shapefile and external data.


The US counties shapefile in `examples/cb_2015_us_county_500k/` is taken from the US Census Bureau's "Cartographic Boundary Shapefiles" and can be downloaded here: [http://www2.census.gov/geo/tiger/GENZ2015/shp/cb_2015_us_county_500k.zip](http://www2.census.gov/geo/tiger/GENZ2015/shp/cb_2015_us_county_500k.zip)

Higher resoltion US county shapefiles can be found here: [https://www.census.gov/geo/maps-data/data/cbf/cbf_counties.html](https://www.census.gov/geo/maps-data/data/cbf/cbf_counties.html)
